## 下载

### Docker

```
docker pull apachecn0/ml-mastery-zh
docker run -tid -p <port>:80 apachecn0/ml-mastery-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g ml-mastery-zh
ml-mastery-zh <port>
# 访问 http://localhost:{port} 查看文档
```
