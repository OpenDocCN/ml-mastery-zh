我觉得这个课程很棒。

本课程不是开发人员对线性代数的介绍，如果这是预期，你可能会落后。

该课程确实假设了线性代数，符号和操作的基础知识的合理流畅性。它并没有预先隐藏这个假设。

如果您对深度学习感兴趣或者更多地了解深度学习方法中使用的线性代数运算，我认为这门课程不是必需的。
